# Changelog

All notable changes to Fost will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial public release of Fost
- CLI tool for AI-powered SDK generation
- Support for OpenAPI (REST API) specifications
- Support for EVM smart contract ABIs (Solidity)
- Basic GraphQL specification support
- TypeScript SDK generation with full type safety
- LLM-powered code generation using OpenAI GPT-4
- Plugin system for extending generation capabilities
- Configuration file support (fost.config.json)
- Comprehensive documentation and examples
- GitHub Actions CI/CD pipeline for testing and publishing

### Changed
- Project restructured for public release
- Documentation updated for general audience

## [0.1.0] - 2026-03-05

### Added
- Initial beta release
- Core CLI functionality
- TypeScript SDK generator
- OpenAPI parser and validator
- EVM ABI analyzer
- LLM integration with fallback strategies
- Test fixtures and examples
- CLI argument parsing and validation

### Known Limitations
- Python, Go, Rust generators are currently in development
- Solana IDL support is experimental
- Plugin system API is subject to change
- Only TypeScript SDK generation is production-ready
